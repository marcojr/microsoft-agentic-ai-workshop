Azure AI Foundry / Microsoft Foundry não é uma ferramenta. É uma stack de produção para agentes: modelos + ferramentas + orquestração + dados + observabilidade + segurança/governança.

A Microsoft descreve o Foundry Agent Service como uma plataforma gerenciada para construir, publicar e escalar agentes, usando modelos do Foundry catalog e frameworks como Microsoft Agent Framework, LangGraph ou código próprio.

O que é essencial para você
1. Foundry como “control plane” de agentes

Você precisa entender o Foundry como o lugar onde a empresa organiza:

modelos
agents
tools
observability
evaluation
tracing
RBAC
governance
policies
monitoring

A Microsoft posiciona o Microsoft Foundry como uma plataforma que unifica agents, models e tools com tracing, monitoring, evaluations, RBAC, networking e policies.

Para entrevista, sua frase é:

“I see Microsoft Foundry as the enterprise control plane for agents, models, tools, evaluation, observability and governance.”

Isso é muito bom.
